Laravelインストール ​

概要 ​

  • Laravelをインストールする

Sailは以下の理由のため非推奨

  • Amazon LinuxなどのRHEL系のOSを使用できない
    • Sail標準のDockerfileのベースイメージがUbuntuで、aptコマンドを使用している
  • Apache、nginxなどのWebサーバーを使用できない
    • Sailは標準でWebサーバーでなくPHPの組み込みサーバーを使用している

Docker用のファイル を作成 ​

  • Laravel用Dockerファイル群のソースをダウンロード

補足

  • 以降の手順は上記例をベースに記載する

Laravelプロジェクト の作成 ​

  • 以下のコマンドを実行
Shell
rm -f /var/www/html/.gitkeep 
composer create-project laravel/laravel /var/www/html

Laravelプロジェクト の初期設定 ​

ファイルの権限設定 ​

  • 以下のコマンドを実行
Shell
chown -R apache:apache /var/www/html
chmod -R 755 /var/www/html

envの初期設定 ​

ローカル環境用envを作成 ​

  • src/.env.example を src/.env.local にリネーム

ローカル環境用envを編集 ​

Shell
# 共通
APP_NAME=(アプリケーション名)
APP_TIMEZONE=Asia/Tokyo
APP_URL=http://localhost:8080
APP_LOCALE=ja
APP_FALLBACK_LOCALE=ja
APP_FAKER_LOCALE=ja_JP

# DB
DB_CONNECTION=mysql
DB_HOST=db
DB_PORT=3306
DB_DATABASE=foo
DB_USERNAME=user
DB_PASSWORD=user

# メール
MAIL_MAILER=smtp
MAIL_HOST=mailpit
MAIL_PORT=1025

ローカル環境用envを適用 ​

  • src/.env.local と src/.env の内容を比較し、.env に差分を反映する

疎通確認 ​

  • http://localhost:8080/ にアクセス
    • Laravelの初期画面が表示されることを確認
